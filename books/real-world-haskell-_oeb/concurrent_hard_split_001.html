Exercises

  1. The Chan type is implemented using MVars. Use MVars to develop a BoundedChan library.

    Your newBoundedChan function should accept an Int parameter, limiting the number of unread items that can be present in a BoundedChan at once.

    If this limit is hit, a call to your writeBoundedChan function must block until a reader uses readBoundedChan to consume a value.

  2. Although we've already mentioned the existence of the strict-concurrency package in the Hackage repository, try developing your own, as a wrapper around the built-in MVar type. Following classic Haskell practice, make your library type safe, so that users cannot accidentally mix uses of strict and nonstrict MVars.
